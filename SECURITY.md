# Security Policy

## Supported Versions

Only the current version (v3.1+) is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 3.1.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT create a public issue. Instead, report it to the IAH Games security team:

1. **Email**: security@iahgames.example (This is a placeholder)
2. **Response**: We aim to respond within 48 hours.
3. **Disclosure**: We will coordinate the disclosure timeline with you.
